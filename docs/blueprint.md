# **App Name**: Barcode Symphony

## Core Features:

- Text Input: Text Input Area: Allow users to input a list of numbers into a multi-line text area.
- Column Barcode Generation: Generate Barcodes (Column): Automatically generate Code128 barcodes in a single column format based on the input list.
- Barcode Clarity Toggle: Interactive Barcode Display: Initially display the barcodes with the code text prominently visible, and toggle to a clear barcode-only view with smaller text upon clicking.
- Grid Barcode Generation: Generate Barcodes (Grid): Display another set of Code128 barcodes in a grid of 3 rows by 10 columns, each distinctly numbered with sufficient margins.
- AI Assisted Format Validation: Format Correction Tool: use the AI to reformat the input so it matches code 128 requirements, if needed.
- Grid Item Numbering: Number the barcode items that will be displayed in the grid layout

## Style Guidelines:

- Primary color: Dark blue (#2E3192) to provide a sense of reliability and precision.
- Background color: Light gray (#F0F2F5) for a clean, professional interface.
- Accent color: Teal (#008080) to highlight interactive elements and call to action.
- Body and headline font: 'Inter' sans-serif font to provide a clean, neutral reading experience.
- Code font: 'Source Code Pro' for code sections, and in the small barcode label.
- Maintain clear margins around barcodes and text areas to ensure readability and visual clarity. Align barcodes uniformly in both column and grid layouts.
- Subtle fade animation on the click of a barcode item.
